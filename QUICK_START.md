## âš ï¸ Common Issues & Quick Fixes

### "404 - Not Found"
**Fix**: 
- Make sure file is named `index.html` (not `super-mario-platformer.html`)
- Check that you selected "main" branch in Pages settings

### "Page Not Loading After 5 Minutes"
**Fix**:
- Hard refresh: `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)
- Check repository is **Public** (not Private)
- Verify GitHub Pages shows "Your site is live"

### "Game Shows But Doesn't Work"
**Fix**:
- Press F12 to open browser console
- Look for JavaScript errors
- Make sure you uploaded the complete `index.html` file
